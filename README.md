# Battery Work Tracker Pro

Ελληνική εφαρμογή για GitHub Pages που αποθηκεύει το ιστορικό εργασίας στον browser.

## Δυνατότητες

- Καταχώρηση ημερομηνίας, πραγματικών ωρών, μπαταριών και σημείωσης.
- Επεξεργασία και διαγραφή καταχωρήσεων.
- Ιστορικό με φίλτρο μήνα, έτους και αναζήτησης.
- Πραγματικά κέρδη: 5€ ανά ώρα + 0,20€ ανά μπαταρία.
- Εικονικά κέρδη: κάθε εργάσιμη μέρα υπολογίζεται ως 4ωρο + 0,20€ ανά μπαταρία.
- Γράφημα ανά μήνα.
- Εξαγωγή σε αρχείο Excel (.xls).
- Εκτύπωση/αποθήκευση σε PDF από τον browser.
- Dark mode.
- PWA εγκατάσταση στο κινητό από το GitHub Pages.

## Ανέβασμα στο GitHub Pages

1. Αποσυμπίεσε το ZIP.
2. Ανέβασε όλα τα αρχεία στο repository σου.
3. Πήγαινε Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: main / root.
6. Άνοιξε το link που θα σου δώσει το GitHub Pages.

## Σημαντικό

Τα δεδομένα αποθηκεύονται στο ίδιο κινητό/browser με localStorage. Αν αλλάξεις συσκευή ή καθαρίσεις τα δεδομένα browser, δεν μεταφέρονται αυτόματα.
