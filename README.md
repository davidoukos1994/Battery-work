# Battery Work - Storage Fix v2

Αυτή η έκδοση διορθώνει κυρίως τη λειτουργία από την Αφετηρία iPhone.

Τι αλλάζει:
- Δεν χρησιμοποιεί standalone PWA αποθήκευση.
- Το εικονίδιο από την Αφετηρία ανοίγει τη σελίδα σαν Safari, ώστε να βλέπει τα ίδια δεδομένα.
- Καθαρίζει παλιό service worker/cache για να μη φορτώνει παλιά έκδοση.

Ανέβασε όλα τα αρχεία στο GitHub και κάνε overwrite.
Μετά άνοιξε τη σελίδα στο Safari και κάνε refresh.
Αν έχεις παλιό εικονίδιο στην Αφετηρία, διέγραψέ το και πρόσθεσέ το ξανά.
