# Battery Work - monthly-backup-2.1.0

Αλλαγές:
- Αφαιρέθηκε το πάνω πλαίσιο με τα συνολικά στοιχεία: Ημέρες, Ώρες, Μπαταρίες, Πραγματικά, Εικονικά.
- Η σελίδα ξεκινά καθαρά με το πλαίσιο «Νέα καταχώρηση».
- Το «Ιστορικό ανά μήνα» μένει όπως ήταν.
- Προστέθηκε προαιρετικός συγχρονισμός GitHub για να βλέπεις τα ίδια δεδομένα σε κινητό και υπολογιστή.

Γιατί δεν φαίνονταν ίδια δεδομένα σε κινητό και υπολογιστή:
Η παλιά έκδοση χρησιμοποιούσε localStorage. Κάθε συσκευή/browser έχει ξεχωριστή τοπική αποθήκευση, άρα δεν μοιράζονται αυτόματα τα δεδομένα.

Για συγχρονισμό GitHub:
1. Ανέβασε όλα τα αρχεία του φακέλου στο GitHub Pages repo σου.
2. Άνοιξε την εφαρμογή.
3. Πήγαινε στο «Συγχρονισμός συσκευών» > «Ρυθμίσεις».
4. Βάλε GitHub owner, repo, branch, path και GitHub token.
5. Το token πρέπει να έχει δικαίωμα Contents: Read and Write στο συγκεκριμένο repo.
6. Πάτησε «Αποθήκευση ρυθμίσεων».

Σημαντικό:
Το GitHub token αποθηκεύεται μόνο στη συσκευή σου μέσα στο browser. Μην το βάζεις σε ξένη ή κοινόχρηστη συσκευή.
